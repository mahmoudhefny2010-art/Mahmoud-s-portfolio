3. **Share with companies:**

## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css` (line 4-17):
```css
--primary-color: #6366f1;      /* Change to your preferred color */
--secondary-color: #ec4899;    /* Change accent color */
```

### Update Content
Edit `index.html` to update:
- Your name and title
- Experience entries
- Skills and education
- Contact information

### Adjust Animations
All animations are in `styles.css`. Look for `@keyframes` sections to modify:
- Speed (change duration values like `0.8s`)
- Effects (modify `transform` and `opacity` values)
